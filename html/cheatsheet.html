üèóÔ∏è Document Structure

The fundamental building blocks of every HTML document.

Basic HTML Template

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

Every HTML page starts with this structure. The DOCTYPE tells browsers this is HTML5.

<head> Element

<head>
    <meta charset="UTF-8">
    <title>My Page</title>
    <link rel="stylesheet" href="styles.css">
    <script src="app.js"></script>
</head>

Contains metadata, links to stylesheets, and scripts.

<body> Element

<body>
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
</body>

Contains all visible content on the page.

üìù Text Elements

Elements for displaying and formatting text content.

Headings

<h1>Main Heading</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>
<h4>Minor Heading</h4>
<h5>Small Heading</h5>
<h6>Smallest Heading</h6>

h1 is the most important, h6 the least. Use only one h1 per page.

Paragraphs

<p>This is a paragraph of text.</p>

<p>This is another paragraph.
   Multiple lines are joined.</p>

Use paragraphs to group related sentences together.

Text Formatting

<strong>Bold text</strong>
<em>Italic text</em>
<mark>Highlighted</mark>
<del>Deleted text</del>
<ins>Inserted text</ins>
<sub>Subscript</sub>
<sup>Superscript</sup>

Use <strong> for importance, <em> for emphasis.

Line Breaks & Rules

<p>Line one<br>Line two</p>

<hr>

<p>Content after horizontal rule</p>

<br> breaks text, <hr> adds a horizontal line.

Blockquote

<blockquote cite="source-url">
    <p>A famous quote here.</p>
    <footer>‚Äî Author Name</footer>
</blockquote>

Use for quoting external sources or long quotes.

Code & Preformatted

<code>const x = 10;</code>

<pre>
  Preformatted
  text preserves
  spacing
</pre>

<code> for inline code, <pre> for code blocks.

üîó Links & Images

Connecting pages and adding visual content.

Links (Anchor Tags)

<!-- External link -->
<a href="https://example.com">Visit Site</a>

<!-- New tab -->
<a href="url" target="_blank">New Tab</a>

<!-- Email link -->
<a href="mailto:hello@example.com">Email</a>

<!-- Page section -->
<a href="#section-id">Jump to Section</a>

The href attribute specifies the destination.

Images

<img src="image.jpg" alt="Description">

<!-- With dimensions -->
<img src="photo.png" 
     alt="A photo" 
     width="300" 
     height="200">

<!-- Lazy loading -->
<img src="big.jpg" 
     alt="Large image" 
     loading="lazy">

Always include alt text for accessibility!

Figure & Caption

<figure>
    <img src="chart.png" alt="Sales chart">
    <figcaption>
        Q4 2024 Sales Data
    </figcaption>
</figure>

Use figure to group images with their captions.

Responsive Images

<picture>
    <source media="(min-width: 800px)" 
            srcset="large.jpg">
    <source media="(min-width: 400px)" 
            srcset="medium.jpg">
    <img src="small.jpg" alt="Responsive">
</picture>

Serve different images based on screen size.

üìã Lists

Organizing content in ordered and unordered lists.

Unordered List

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Bullet points for items without a specific order.

Ordered List

<ol>
    <li>Step one</li>
    <li>Step two</li>
    <li>Step three</li>
</ol>

<!-- Start from different number -->
<ol start="5">
    <li>Item five</li>
</ol>

Numbered items for sequential content.

Description List

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

Term-definition pairs, great for glossaries.

Nested Lists

<ul>
    <li>Frontend
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </li>
    <li>Backend</li>
</ul>

Lists can be nested inside other lists.

üìä Tables

Displaying data in rows and columns.

Basic Table

<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>City</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alice</td>
            <td>28</td>
            <td>London</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>32</td>
            <td>Paris</td>
        </tr>
    </tbody>
</table>

Use <th> for headers, <td> for data cells.

Spanning Cells

<!-- Span 2 columns -->
<td colspan="2">Wide cell</td>

<!-- Span 3 rows -->
<td rowspan="3">Tall cell</td>

Merge cells across rows or columns.

Table Caption

<table>
    <caption>Employee Directory</caption>
    <thead>...</thead>
    <tbody>...</tbody>
</table>

Add a title to describe the table content.

üìÆ Forms

Collecting user input and data.

Basic Form

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Submit</button>
</form>

Always use <label> elements with the for attribute.

Text Inputs

<input type="text" placeholder="Name">
<input type="email" placeholder="Email">
<input type="password" placeholder="Password">
<input type="tel" placeholder="Phone">
<input type="url" placeholder="Website">
<input type="search" placeholder="Search...">

Different types provide validation and mobile keyboards.

Other Input Types

<input type="number" min="0" max="100">
<input type="range" min="0" max="100">
<input type="date">
<input type="time">
<input type="color">
<input type="file" accept="image/*">

HTML5 provides many specialized input types.

Checkbox & Radio

<!-- Checkboxes (multiple) -->
<input type="checkbox" id="html" name="skills" value="html">
<label for="html">HTML</label>

<!-- Radio buttons (single) -->
<input type="radio" id="yes" name="answer" value="yes">
<label for="yes">Yes</label>
<input type="radio" id="no" name="answer" value="no">
<label for="no">No</label>

Same name groups radio buttons together.

Select Dropdown

<select name="country">
    <option value="">Select...</option>
    <option value="uk">United Kingdom</option>
    <option value="us">United States</option>
    <option value="fr">France</option>
</select>

Dropdown for selecting from predefined options.

Textarea

<textarea name="message" 
          rows="4" 
          cols="50" 
          placeholder="Your message...">
</textarea>

Multi-line text input for longer content.

üèõÔ∏è Semantic Elements

Elements that describe the meaning of content.

Page Layout

<header>Site header, logo, navigation</header>

<nav>Navigation links</nav>

<main>Main content (one per page)</main>

<aside>Sidebar, related content</aside>

<footer>Footer, copyright, links</footer>

Semantic elements improve accessibility and SEO.

Content Sections

<article>
    Self-contained content like 
    blog posts or news articles
</article>

<section>
    Thematic grouping of content
</section>

Articles can be shared independently; sections group related content.

Other Semantic Elements

<time datetime="2024-12-25">
    December 25, 2024
</time>

<address>
    Contact information
</address>

<details>
    <summary>Click to expand</summary>
    Hidden content here
</details>

Specialized elements for specific types of content.

üè∑Ô∏è Meta Tags

Page metadata for browsers and search engines.

Essential Meta Tags

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Page description for SEO">
<meta name="keywords" content="html, css, web development">
<meta name="author" content="Your Name">

Always include charset and viewport for proper rendering.

Open Graph (Social)

<meta property="og:title" content="Page Title">
<meta property="og:description" content="Description">
<meta property="og:image" content="image.jpg">
<meta property="og:url" content="https://...">

Controls how pages appear when shared on social media.

Favicon

<link rel="icon" type="image/x-icon" 
      href="/favicon.ico">

<link rel="apple-touch-icon" 
      href="/apple-icon.png">

The small icon shown in browser tabs.
