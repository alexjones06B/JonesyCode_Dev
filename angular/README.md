# Angular Learning Module

## Build and Run

```bash
# Install dependencies
npm install

# Build TypeScript files
npm run build

# Watch for changes
npm run watch

# Serve locally
npm run serve
```

## Files

- `index.html` - Main course overview page
- `cheatsheet.html` - Angular quick reference guide
- `walkthroughs/` - Step-by-step learning exercises
- `scripts/` - TypeScript walkthrough definitions
- `styles/` - CSS styling

## Structure

1. **Angular Basics** - Modules, CLI, project structure
2. **Components** - Building reusable UI blocks
3. **Templates & Data Binding** - Connecting data to views
4. **Services & DI** - Sharing logic across components

## Prerequisites

Complete the TypeScript course first!
