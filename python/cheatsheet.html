üéØ Basics

Fundamental Python syntax and concepts.

Print Output

print("Hello, World!")
print("Name:", name)
print(f"Hello, {name}!")

Display output to the console.

Variables

name = "Alex"        # String
age = 25             # Integer
price = 19.99        # Float
is_active = True     # Boolean

No type declaration needed. Python infers types.

Comments

# This is a single-line comment

"""
This is a
multi-line comment
(docstring)
"""

Use # for single line, triple quotes for multi-line.

User Input

name = input("Enter name: ")
age = int(input("Enter age: "))

input() always returns a string. Convert as needed.

Type Checking

type(42)          # 
type("hello")     # 
isinstance(42, int)  # True

Check types with type() or isinstance().

Type Conversion

int("42")     # String to int ‚Üí 42
str(42)       # Int to string ‚Üí "42"
float("3.14") # String to float ‚Üí 3.14
list("abc")   # String to list ‚Üí ['a','b','c']

Convert between types with built-in functions.

üìù Strings

Text manipulation and formatting.

String Creation

single = 'Hello'
double = "World"
multi = """Multiple
lines"""
raw = r"C:\new\folder"  # Raw string

Use single or double quotes. Triple quotes for multi-line.

F-Strings (Formatting)

name = "Alex"
age = 25
f"Hello, {name}!"
f"Next year: {age + 1}"
f"Price: ${price:.2f}"

Modern way to format strings. Use f prefix.

String Methods

s = "Hello World"
s.upper()       # "HELLO WORLD"
s.lower()       # "hello world"
s.strip()       # Remove whitespace
s.split(" ")    # ["Hello", "World"]
s.replace("l", "x")  # "Hexxo Worxd"

Strings are immutable - methods return new strings.

String Slicing

s = "Python"
s[0]      # 'P' (first char)
s[-1]     # 'n' (last char)
s[0:2]    # 'Py'
s[2:]     # 'thon'
s[::-1]   # 'nohtyP' (reversed)

Access characters and substrings by index.

String Checks

s.startswith("Py")  # True
s.endswith(".py")   # False
s.isdigit()         # False
s.isalpha()         # True
"th" in s           # True

Boolean methods for string validation.

Join & Split

words = ["Hello", "World"]
" ".join(words)  # "Hello World"

text = "a,b,c"
text.split(",")  # ["a", "b", "c"]

Convert between strings and lists.

üî¢ Numbers

Numeric types and mathematical operations.

Number Types

integer = 42
float_num = 3.14
complex_num = 3 + 4j
big_num = 1_000_000  # Readable

Python has int, float, and complex number types.

Arithmetic Operators

5 + 3    # 8  (add)
5 - 3    # 2  (subtract)
5 * 3    # 15 (multiply)
5 / 3    # 1.666... (divide)
5 // 3   # 1  (floor divide)
5 % 3    # 2  (modulo/remainder)
5 ** 3   # 125 (power)

Standard math operators plus floor division and power.

Assignment Operators

x = 5
x += 3   # x = x + 3 ‚Üí 8
x -= 2   # x = x - 2 ‚Üí 6
x *= 2   # x = x * 2 ‚Üí 12
x /= 3   # x = x / 3 ‚Üí 4.0

Shorthand for modify-and-assign.

Math Functions

abs(-5)      # 5
round(3.7)   # 4
min(1, 2, 3) # 1
max(1, 2, 3) # 3
sum([1,2,3]) # 6

import math
math.sqrt(16)    # 4.0
math.floor(3.7)  # 3
math.ceil(3.2)   # 4

Built-in and math module functions.

Comparison Operators

5 == 5   # True (equal)
5 != 3   # True (not equal)
5 > 3    # True
5 < 3    # False
5 >= 5   # True
5 <= 3   # False

Compare values, returns boolean.

Random Numbers

import random

random.random()        # 0.0 to 1.0
random.randint(1, 10)  # 1 to 10
random.choice([1,2,3]) # Pick one
random.shuffle(mylist) # Shuffle in-place

Generate random numbers and selections.

üìã Lists

Ordered, mutable sequences of items.

Creating Lists

empty = []
numbers = [1, 2, 3, 4, 5]
mixed = [1, "two", 3.0, True]
nested = [[1, 2], [3, 4]]
from_range = list(range(5))

Lists can contain any types, including other lists.

Accessing Items

nums = [10, 20, 30, 40, 50]
nums[0]     # 10 (first)
nums[-1]    # 50 (last)
nums[1:3]   # [20, 30]
nums[::2]   # [10, 30, 50] (every 2nd)

Zero-indexed. Negative indexes count from end.

Modifying Lists

nums = [1, 2, 3]
nums.append(4)      # [1,2,3,4]
nums.insert(0, 0)   # [0,1,2,3,4]
nums.extend([5,6])  # [0,1,2,3,4,5,6]
nums[0] = 99        # Change first item
nums.remove(99)     # Remove by value
del nums[0]         # Remove by index

Lists are mutable - can be modified in-place.

List Methods

nums = [3, 1, 4, 1, 5]
nums.sort()         # [1,1,3,4,5]
nums.reverse()      # [5,4,3,1,1]
nums.pop()          # Remove & return last
nums.count(1)       # 2 (occurrences)
nums.index(4)       # Find position
len(nums)           # Length

Common list operations and methods.

List Operations

[1, 2] + [3, 4]  # [1,2,3,4] concat
[1, 2] * 3       # [1,2,1,2,1,2]
3 in [1, 2, 3]   # True
3 not in [1, 2]  # True

Concatenate, repeat, and check membership.

Copying Lists

original = [1, 2, 3]
# Shallow copies:
copy1 = original.copy()
copy2 = list(original)
copy3 = original[:]

# Deep copy (nested):
import copy
deep = copy.deepcopy(nested)

‚ö†Ô∏è Assignment (=) creates reference, not copy!

üìñ Dictionaries

Key-value pairs for fast lookups.

Creating Dictionaries

empty = {}
person = {
    "name": "Alex",
    "age": 25,
    "city": "London"
}
from_pairs = dict([("a", 1), ("b", 2)])

Keys must be immutable (strings, numbers, tuples).

Accessing Values

person["name"]       # "Alex"
person.get("name")   # "Alex"
person.get("job", "N/A")  # With default

# Safe - returns None if missing:
person.get("missing")  # None
# Unsafe - raises KeyError:
person["missing"]  # KeyError!

Use .get() for safe access with optional default.

Modifying Dictionaries

person["age"] = 26        # Update
person["job"] = "Dev"     # Add new
del person["city"]        # Delete
person.pop("job")         # Remove & return
person.update({"a": 1})   # Merge

Dictionaries are mutable.

Dictionary Methods

person.keys()    # dict_keys([...])
person.values()  # dict_values([...])
person.items()   # dict_items([...])
len(person)      # Number of pairs
"name" in person # True

Iterate over keys, values, or both.

Iterating

# Keys only
for key in person:
    print(key)

# Keys and values
for key, value in person.items():
    print(f"{key}: {value}")

Loop through dictionary contents.

Dictionary Comprehension

# Square numbers
squares = {x: x**2 for x in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

# Filter
adults = {k:v for k,v in ages.items() 
          if v >= 18}

Create dictionaries with concise syntax.

üîÄ Control Flow

Conditionals and loops.

If / Elif / Else

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

Conditions are checked in order. First True wins.

Ternary Operator

status = "adult" if age >= 18 else "minor"

# Same as:
if age >= 18:
    status = "adult"
else:
    status = "minor"

One-line conditional assignment.

For Loop

# Iterate over list
for fruit in fruits:
    print(fruit)

# With index
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# Range
for i in range(5):      # 0-4
for i in range(2, 5):   # 2-4
for i in range(0, 10, 2): # 0,2,4,6,8

Iterate over any sequence or range.

While Loop

count = 0
while count < 5:
    print(count)
    count += 1

# With else (runs if no break)
while condition:
    # code
else:
    # runs when condition becomes False

Loop while condition is True.

Break & Continue

# break - exit loop entirely
for i in range(10):
    if i == 5:
        break  # Stops at 5

# continue - skip to next iteration
for i in range(5):
    if i == 2:
        continue  # Skips 2
    print(i)  # 0,1,3,4

Control loop execution flow.

Logical Operators

True and False  # False
True or False   # True
not True        # False

# Short-circuit evaluation
x = a or default_value
y = condition and result

Combine conditions with and, or, not.

‚ö° Functions

Reusable blocks of code.

Basic Function

def greet(name):
    """Greet a person."""
    return f"Hello, {name}!"

# Call it
message = greet("Alex")

Use def keyword. Docstring describes function.

Default Parameters

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

greet("Alex")           # "Hello, Alex!"
greet("Alex", "Hi")     # "Hi, Alex!"

Parameters with defaults are optional.

Keyword Arguments

def create_user(name, age, city):
    return {"name": name, "age": age}

# Call with keywords (any order):
create_user(age=25, name="Alex", city="LA")

Explicitly name arguments when calling.

*args and **kwargs

# *args - variable positional args
def sum_all(*numbers):
    return sum(numbers)
sum_all(1, 2, 3, 4)  # 10

# **kwargs - variable keyword args
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")
print_info(name="Alex", age=25)

Accept arbitrary number of arguments.

Multiple Return Values

def get_stats(numbers):
    return min(numbers), max(numbers)

# Unpack the tuple
smallest, largest = get_stats([1,2,3,4,5])
# smallest=1, largest=5

Return multiple values as a tuple.

Lambda Functions

# Anonymous inline function
square = lambda x: x ** 2
square(5)  # 25

# Common with map, filter, sort
nums = [3, 1, 4, 1, 5]
sorted(nums, key=lambda x: -x)
list(map(lambda x: x*2, nums))
list(filter(lambda x: x > 2, nums))

Short, anonymous functions for simple operations.

üîÑ Comprehensions

Concise ways to create collections.

List Comprehension

# Basic
squares = [x**2 for x in range(5)]
# [0, 1, 4, 9, 16]

# With condition
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]

# Transform
upper = [s.upper() for s in words]

Create lists in a single line.

Dictionary Comprehension

squares = {x: x**2 for x in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

# From two lists
keys = ["a", "b", "c"]
vals = [1, 2, 3]
d = {k: v for k, v in zip(keys, vals)}

Create dictionaries in a single line.

Set Comprehension

unique_lengths = {len(word) for word in words}

# From list with duplicates
nums = [1, 2, 2, 3, 3, 3]
unique = {x for x in nums}  # {1, 2, 3}

Create sets with automatic deduplication.

Generator Expression

# Like list comp but with ()
gen = (x**2 for x in range(1000000))

# Memory efficient - generates on demand
for val in gen:
    print(val)

Lazy evaluation - great for large data.

üìÅ File I/O

Reading and writing files.

Reading Files

# Read entire file
with open("file.txt", "r") as f:
    content = f.read()

# Read lines into list
with open("file.txt", "r") as f:
    lines = f.readlines()

# Read line by line
with open("file.txt", "r") as f:
    for line in f:
        print(line.strip())

Use 'with' for automatic file closing.

Writing Files

# Write (overwrites)
with open("file.txt", "w") as f:
    f.write("Hello, World!")

# Append
with open("file.txt", "a") as f:
    f.write("\nNew line")

# Write multiple lines
with open("file.txt", "w") as f:
    f.writelines(["Line 1\n", "Line 2\n"])

"w" overwrites, "a" appends to existing content.

JSON Files

import json

# Write JSON
with open("data.json", "w") as f:
    json.dump(my_dict, f, indent=2)

# Read JSON
with open("data.json", "r") as f:
    data = json.load(f)

# String conversion
json_str = json.dumps(my_dict)
my_dict = json.loads(json_str)

dump/load for files, dumps/loads for strings.

Path Operations

from pathlib import Path

path = Path("folder/file.txt")
path.exists()      # Check if exists
path.is_file()     # Is it a file?
path.is_dir()      # Is it a directory?
path.read_text()   # Read content
path.write_text("content")  # Write

Modern path handling with pathlib.

üèóÔ∏è Classes

Object-oriented programming.

Basic Class

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hi, I'm {self.name}"

# Create instance
person = Person("Alex", 25)
person.greet()  # "Hi, I'm Alex"

__init__ is the constructor. self refers to instance.

Inheritance

class Student(Person):
    def __init__(self, name, age, grade):
        super().__init__(name, age)
        self.grade = grade
    
    def study(self):
        return f"{self.name} is studying"

student = Student("Bob", 20, "A")
student.greet()  # Inherited method
student.study()  # Own method

Use super() to call parent methods.

Class vs Instance Attributes

class Dog:
    species = "Canis familiaris"  # Class attr
    
    def __init__(self, name):
        self.name = name  # Instance attr

dog1 = Dog("Rex")
dog2 = Dog("Buddy")
Dog.species    # Shared by all
dog1.name      # Unique to instance

Class attributes are shared by all instances.

Special Methods

class Point:
    def __init__(self, x, y):
        self.x, self.y = x, y
    
    def __str__(self):  # print()
        return f"({self.x}, {self.y})"
    
    def __repr__(self):  # repr()
        return f"Point({self.x}, {self.y})"
    
    def __eq__(self, other):  # ==
        return self.x == other.x and self.y == other.y

Dunder methods customize class behavior.
